# Support

Questions are welcome. If you are new to open source or MCP tooling, you are not expected to know everything.

## Where to Ask

- **Bug**: open a bug report using the issue template
- **Feature idea**: open a feature request using the issue template
- **General help**: open an issue and include `[QUESTION]` in the title

## What to Include

To help maintainers respond quickly, please include:

- What you are trying to do
- What happened vs what you expected
- OS, Python version, and MCP client
- Relevant logs or error messages (redact secrets)
- What you already tried

## Response Style

Maintainers aim for responses that are respectful, practical, and beginner-friendly.

If anything is unclear, ask follow-up questions. That is encouraged.
